Details / Summary

What is rek-room?

rek-room is a classless CSS design system that styles native HTML elements using design tokens. It's built for readability, accessibility, and easy customization via CSS custom properties.

How do I customize the theme?

Override any of the CSS custom properties defined in tokens/core.css. You can change colors, spacing, typography, border radii, and more without touching the component stylesheets.

Does it support dark mode?

Yes. rek-room uses color-scheme and light-dark() functions throughout, so toggling dark mode is as simple as setting color-scheme: dark on the root element.

Question 1: What is rek-room?

A zero-dependency CSS starting point for styling HTML primitives and simple components.

Question 2: Does it require JavaScript?

No. Everything is pure CSS — no JavaScript required for any component.

Question 3: How do I customize it?

Override any CSS custom property. Because the system uses @layer, your overrides automatically win.
